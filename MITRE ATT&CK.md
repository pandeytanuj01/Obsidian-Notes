#framework 

It is a framework which consists of **TTPs** used by adversaries in real world. It is used by organizations to understand attack behavior, build detections, map security coverage and improve incident response.

**TTPs** refer to Tactics, Techniques and Procedures.

**Tactics** refer to the goal of the attacker. 

For e.g.

Initial access: How do i get inside?
Execution: How do i run malware?
Persistence: How do i stay after reboot?
Lateral Movement: How do i move to another PC?
Exfiltration: How do i steal data?

**Technique** refers to how exactly the attacker achieve that goal.'

For e.g.

Gained Initial Access using Phishing.
Ran the malware using PowerShell.
Gained persistence via Scheduled Task.

**Procedures** refer to how exactly these techniques were performed in the attack. Real world examples of adversaries performing these techniques.
