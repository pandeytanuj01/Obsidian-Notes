#framework

This framework is used to defined the attack chain followed by adversaries to perform a successful attack. It is used by organizations to identify and prevent cyber attacks.

It has a total of 7 stages:

### Reconnaissance: Gathering information related to the target
### Weaponization: Creating a payload with exploit and a backdoor

### Delivery: Delivering the payload to the target via mediums such as email, web, USB, etc.

### Exploitation: Exploiting the vulnerability in the system to gain access to the target's system

### Installation: Installing malware on the target

### Command and Control: Establishing a communication channel between target and attacker's system to access the system remotely and execute commands

### Actions on Objective: With remote access to the target system, attacker can accomplish their goals, such as data exfiltration, deploying ransomware, privilege escalation, etc.
